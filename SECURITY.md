# Security Policy

## Supported Versions

OpenTristam tracks security fixes on the default branch.

## Reporting a Vulnerability

Please report vulnerabilities privately by opening a GitHub Security Advisory:

- Go to the repository "Security" tab
- Choose "Report a vulnerability"
- Include reproduction steps and impact details

If private reporting is unavailable, open a private issue with maintainers tagged and avoid public disclosure details until a fix is ready.

## Response Process

- Initial triage target: within 5 business days
- Confirmed issues are patched on the default branch first
- Public disclosure happens after a fix is available
